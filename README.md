## ✨ Features

- 📞 **One-tap Call** — clicking the phone number opens your dialer directly
- 📧 **One-tap Email** — clicking the email launches your mail app
- 🌐 **College Link** — direct link to the institution website
- 📊 **Animated Skill Bars** — bars animate in as you scroll
- 🟢 **Live Enrollment Badge** — pulsing dot shows currently enrolled status
- 📱 **Fully Responsive** — works on mobile, tablet & desktop
- ⚡ **Zero Dependencies** — pure HTML / CSS / JS, no libraries needed

## 🚀 Deploy on GitHub Pages (Make it a Live Website!)

1. Upload `resume_rafe.html` and this `README.md` to a new GitHub repo
2. Rename `resume_rafe.html` → **`index.html`** *(important!)*
3. Go to your repo → **Settings** → **Pages**
4. Under **Branch**, select `main` → `/ (root)` → click **Save**
5. Wait ~60 seconds, then visit:
   ```
   https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
   ```
6. Update the **Live Preview** badge link above with your actual URL!
